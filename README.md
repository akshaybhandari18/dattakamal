## 🚀 How to Deploy (Free on GitHub Pages)

### One-time setup (5 minutes):

1. **Go to** [github.com](https://github.com) and sign in (or create a free account)
2. **Click** the green **"New"** button (top left) to create a new repository
3. **Repository name:** `dattakamal`
4. **Keep it Public** (required for free GitHub Pages)
5. **Click** "Create repository"
6. **Upload files:** Click "uploading an existing file" link, then drag and drop `index.html` and `card.html`
7. **Click** "Commit changes"

### Enable GitHub Pages:

8. Go to your repository **Settings** tab
9. Click **Pages** in the left sidebar
10. Under "Source", select **Deploy from a branch**
11. Select branch: **main**, folder: **/ (root)**
12. Click **Save**
13. Wait 1-2 minutes, then your site is live!

### Your URLs will be:
- **Website:** `https://yourusername.github.io/dattakamal/`
- **Visiting Card:** `https://yourusername.github.io/dattakamal/card.html`

## 🔄 How to Update the QR Code (After Creating Google Form)

1. Go to your repository on GitHub
2. Click on `card.html`
3. Click the **pencil icon** (Edit) at the top right
4. Find this line near the bottom:
   ```
   const QR_URL = "https://wa.me/919423217605";
   ```
5. Replace it with your Google Form link:
   ```
   const QR_URL = "https://forms.gle/XXXXXXXXXX";
   ```
6. Click **"Commit changes"**
7. Wait 1 minute — QR code is updated!
